Tutorial Rápido:

Para usar o caixa eletrônico:

  1. Insira seu cartão no leitor de cartão.
  2. Digite seu PIN usando o teclado numérico.
  3. Selecione a operação desejada (saque, depósito, etc.).
  4. Siga as instruções na tela para concluir a transação.
